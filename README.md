# 🎓 Student Performance Predictor (ML + Flask)

## 📌 Project Overview
Student Performance Predictor is a full-stack Machine Learning web application that predicts a student's academic performance based on input features like study time, failures, and absences.

This project is built using **Python, Flask, HTML, CSS, and Machine Learning** and includes authentication, graphs, and a modern UI.

---

## 🚀 Features

✔ User Login System 🔐  
✔ Machine Learning Prediction 🤖  
✔ Interactive Web Interface 🌐  
✔ Data Visualization (Graphs) 📊

## ⚙️ How to Run Locally

1. Clone the repository:
```
git clone https://github.com/your-username/student-performance-predictor.git
```

2. Go to project folder:
```
cd student-performance-predictor
```

3. Install dependencies:
```
pip install -r requirements.txt
```

4. Run the project:
```
python app.py
```

5. Open in browser:
```
http://127.0.0.1:5000/
```
